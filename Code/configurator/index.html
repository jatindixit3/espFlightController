Ports

This board's UART/I2C/RMT pin assignments are fixed at compile time (see Code/firmware/include/config.h and Electronics/HARDWARE.md) rather than software-remappable like a multi-UART F4/F7 - there's only one MPU6050, one SBUS input, one ESC telemetry wire, and one motor output set, wired to specific pins. Nothing here is editable; it's shown so you know what's using what.

Write sends changes to RAM immediately; Save persists across reboots.

Motor Test — QuadX

Remove propellers before using this tab. Motor test only works while disarmed, but the motors WILL spin. Test throttle is capped at 50% in firmware for safety; you never need more than a gentle spin to check position and direction.

Click a motor to spin it (front is up). Click it again, or click another, to switch. Only one motor spins at a time.

Motor Reordering

If a motor spins in the wrong position during the test, remap which physical output (the ESC signal wire, D0–D3) drives each QuadX position. This applies to real flight too, not just the test.

Motor Direction

If a motor spins the wrong way, reverse it here. This sends the real DShot spin-direction command to that ESC and saves it — props off, disarmed only. The diagram arrows show the direction the firmware's yaw mixing expects for each position; make each motor match its arrow.

Failsafe

This firmware uses the simplest-safe failsafe behavior: if the SBUS link is lost for longer than the timeout below, it disarms and cuts all motors immediately - no staged descent (there's no GPS/baro on this build to do anything smarter with).

Blackbox

Logs to a raw flash region and restarts from the beginning every boot (no persistent ring buffer across power cycles) - download and save a log before your next flight if you want to keep it.

CLI

This is a read-only settings dump, not an interactive command shell - the firmware doesn't implement a text command parser (that's real added scope beyond the binary MSP-style protocol everything else here uses). Use the other tabs to change values; use this to grab a full text snapshot for backup/sharing.
